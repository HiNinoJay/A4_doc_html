INFO

根据你在正式安装教程页选择的方安装式，选择对应的更新方式。

npm方式 ​

命令行进入到你的博客网站根目录下 执行命令：

shell
npm install hexo-theme-a4@latest

git方式 ​

命令行进入到themes/A4目录下 执行命令:

shell
git pull

DANGER

以上两种方式更新后，请去该github页面查看最新的版本声明，作者会告诉配置文件有无新增/修改。

如果有，则需要复制最新的该文件对应到你的_config.a4.yml新增/修改的部分，以体验A4最新的功能。
